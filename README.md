## ğŸš€ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) 18+ and npm

### Quick Start

```bash
# Clone
git clone https://github.com/thehans-mwe/CarbonIQ.git && cd CarbonIQ

# Install
npm install

# Run development server
npm run dev
```

> App launches at `http://localhost:5173`

### Environment Variables <sub>(optional)</sub>

```env
CARBON_API_KEY=your_carbon_interface_api_key
OPENAI_API_KEY=your_openai_api_key
```

<details>
<summary><b>ğŸ’¡ Works without API keys</b></summary>
<br />

Without keys, the calculator uses **accurate offline calculations** powered by:
- **EPA 2024** emission factors for transportation & energy
- **DEFRA 2024** for flights
- **IPCC AR6** for dietary emissions

AI recommendations are replaced with curated offline tips.
